Class Input

Objet contenant comme attribut une balise html "input" (non accessible hors de l'objet). Celle-ci est manipulable par les différentes méthodes fournis. Cet objet permet de garder le code des autres objets intact si l'attribut de l'Objet Input venait à changer. De plus, cette objet peut facilement être réutilisé dans d'autre application javascript manipulant une page html et un fichier css.
Defined in: affichage_partie.js.
